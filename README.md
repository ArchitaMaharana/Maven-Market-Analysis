# Maven Market Analysis – Power BI Project

Maven Market is a multinational grocery retain chain.
This project is a comprehensive analysis of Maven Market data using Power BI to identify business trends and support data driven decision making. It covers multiple business areas including sales, orders, profits, returns, customers, and store performance. The data spans from 1997 to 1998 and is visualized using interactive and insightful dashboards.

## Objective
Analyze overall sales,revenue,profit,transaction trends and identify top performing products, brands,and product categories.
To extract meaningful insights from historical business data to support decision-making across key operational areas of Maven Market.
